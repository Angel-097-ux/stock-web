# ⚡ ANGEL-STOCK // N.PEDIDOS

Sistema de gestión de inventario de alto rendimiento desarrollado para el control logístico de mercadería. Esta aplicación permite la sincronización en tiempo real entre múltiples dispositivos (Mobile/Desktop) utilizando una arquitectura distribuida basada en Firebase.

## 🚀 Características Principales

* **Real-time Sync:** Actualización instantánea de stock mediante Firebase Realtime Database.
* **Diseño Futurista:** Interfaz Cyberpunk optimizada para visibilidad en entornos de trabajo.
* **Gestión Inteligente:** Filtro de búsqueda avanzado por SKU, artículo o color.
* **Cómputo Automático:** Cálculo automático de valor de inventario y total de artículos.
* **Multi-Plataforma:** Compatible con navegadores modernos y operable como PWA (Progressive Web App).

## 🛠️ Stack Tecnológico

* **Frontend:** HTML5, CSS3 (Custom Variables & Keyframes).
* **Lógica:** JavaScript (Vanilla JS - ES6 Modules).
* **Backend:** Firebase Realtime Database.
* **Hosting:** GitHub Pages.
